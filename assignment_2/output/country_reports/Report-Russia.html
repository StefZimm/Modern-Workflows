Report on Russia

Introduction

This project uses data from the European Value Study (EVS) and aims to examine attitudes towards gender roles and immigration for European countries. In the project report two variables on gender role and migration are descriptively examined and a first OLS regression is created. In a further step, the project report will not only provide first results for Europe, but also additional reports will be prepared, which will help to identify possible differences within the EU.

Data Preparation

During data preparation, the required EVS datasets (ZA7500) are unzipped, downloaded and prepared for regression and descriptive analysis. Furthermore, the data set is reduced to the research-relevant variables.

OLS Regression on v72 (Model 1) and v80 (Model 2)

The coefficient Estimate contains six rows; the first one is the intercept. The intercept, in both models, is the expected value 2.4961331(model1) 2.8046314 (model2) of v72/v80 for agreeing to the statement if we consider the independent variables as 0. The next rows describe the Coefficients, or in both models, the effect of the predictors. The slope term of sex in our model is saying if a change of gender to female happens, the y variable increases/decreases by 0.0017626(model1) 0.0183693 (model2). The p-value of this coefficient is 0.9647438 (model1) and 0.7196009 (model2). A small p-value indicates that it is unlikely we will observe a relationship between the predictor and response (v72 or v80) variables due to chance. Typically, a p-value of 0.05 or less is a good cut-off point. Three stars represent a highly significant p-value. Consequently, a small p-value for the intercept and the coefficients indicates that we can reject the null hypothesis which allows us to conclude that there is a relationship between predictor and response.
